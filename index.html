Frequently Asked Questions

Find answers to common questions about CSV to Google Sheets.

Is my data secure when using this extension? +

Absolutely! All data processing happens locally in your browser. Your CSV data never leaves your computer and is not sent to any server. The extension uses the standard clipboard API that's built into your browser.

What format does this extension use? +

The extension converts CSV data to tab-delimited format, which is the most reliable format for pasting into Google Sheets and other spreadsheet applications.

Can I use this with Excel or other spreadsheet applications? +

Yes! While designed with Google Sheets in mind, the formatted data works perfectly with Microsoft Excel, LibreOffice Calc, and virtually any other spreadsheet application that accepts tab-delimited data.

Is there a limit to how much data I can process? +

The extension is optimized to handle large datasets efficiently. However, for very large files (over 100,000 rows), you might experience some performance slowdown. In such cases, consider processing your data in chunks.

Does this extension require any special permissions? +

The extension requires minimal permissions - only access to the clipboard for copying the formatted data. It doesn't need access to your tabs, browsing history, or any other sensitive information.
